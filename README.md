# Rekna

Rekna is a sheet-style calculator which is planned to release for all major desktop operating systems. You type expressions in plain text and Rekna evaluates each row beside the editor.

## Supported Features

- Arithmetic with word operators such as `plus`, `minus`, `times`, `with`, and `without`
- Variables, labels, comments, headings, `prev`, `sum`, `avg`, and blank-line calculation blocks
- Unit conversion with `in`, `into`, `as`, and `to`
- Mixed-unit expressions like `1 meter 20 cm in cm`
- Currency conversion and arithmetic for common ISO codes and symbols such as `USD`, `EUR`, `CAD`, `$`, and `€`
- Time-zone conversion for expressions like `Berlin time`, `time in Madrid`, and `2026-03-22 2:30 pm HKT in Berlin`
- Core math functions in both standard and shorthand syntax, including `sqrt`, `cbrt`, `round`, `fact`, `sin`, `log`, `root`, and `fromunix`
- CSS-oriented units `px` and `em`, with configurable `ppi` and `em` values

## Examples

```text
Price: $30 in EUR
Length: 1 meter 20 cm in cm
Rounded: round 3.45
Time: 2026-03-22 2:30 pm HKT in Berlin
```
